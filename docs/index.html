Features

Everything your research group needs

ProjectHive brings project management, communication, and file sharing together in one place — backed by the reliability of Git.

Kanban Board

Drag-and-drop task management with Todo, Doing, Done, and Blocked columns. Assign members, set due dates, and add labels.

Team Messages

Real-time messaging with @mentions, #task references, emoji reactions, threaded replies, pinning, and labels.

Document Sharing

Upload files and share links in a searchable document library. Attach files directly from messages.

Timeline

Activity feed with commit history, keyword badges, and smart filtering that hides system-generated commits.

Notifications

In-app notification center for @mentions and task assignments, with click-to-navigate and read/unread tracking.

Cross-page Navigation

Click #task references in messages to jump to the board. Click related messages in tasks to scroll right to them.

How It Works

Your data lives in GitHub

ProjectHive stores everything — tasks, messages, docs, config — as JSON files in a GitHub private repo. No server to host. Full version control. Access managed by GitHub permissions.

1

Sign in with GitHub

Authenticate with your personal access token. ProjectHive connects to your GitHub account securely.

2

Create or join a project

Create a new project (auto-generates a private repo) or join an existing one. Invite collaborators by username.

3

Collaborate

Manage tasks, send messages, share files — everything syncs to GitHub. Your team sees updates in real time.

4

Stay in control

Full git history of every change. Switch themes, filter views, and customize your workspace.
